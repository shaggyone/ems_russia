EmsCalculator
=============

EMSCalculator - gem for spree. For calculating delivery price in spree.

Developed with ruby_1.9.2, but must work with some other. 

Example
=======

    gem 'ems_russia', :git => "git://github.com/shaggyone/ems_russia.git", :require => "ems_russia"
